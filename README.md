# Python CLI Habit Streak Tracker

This example demonstrates a simple command-line habit tracker in Python. It allows users to add habits, mark them as complete for the current day, and view their progress. The core technical concept illustrated is the calculation and display of a "streak" – the number of consecutive days a habit has been completed – which serves as a motivational element beyond a basic checklist.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Interact with the menu to add habits, mark them complete, and view streaks.
